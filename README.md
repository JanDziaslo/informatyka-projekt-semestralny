## Tresc z moodla

Dany jest plik dane.txt z 9999 liczbami z zakresu -100000 do 100000.

Napisz program testujący wszystkie poznane na lekcjach metody sortowania.

Ciąg liczb powinien być posortowany określoną metodą, a posortowane liczby zapisuje do odpowiedniego pliku wynikowego.

Za początku pliku wynikowego dopisz 2 wiersze: złożoność obliczeniową danego algorytmu oraz czas wykonania sortowania.

Program napisz z podziałem na funkcje.

Nie zapomnij o walidacji programu.

Przypomnienie: Zapis i odczyt z pliku
